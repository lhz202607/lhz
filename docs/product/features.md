# 古董局中局·十二兽首 — 产品功能

## 游戏简介

改编自马伯庸同名小说的桌游网站，支持 6-8 人同时通过房间码联机对局。

## 核心功能

### 1. 房间联机系统
- 房主创建房间生成 5 位邀请码
- 其他玩家输入邀请码加入
- 实时同步房间状态（玩家进出、准备状态）
- 房主控制游戏开始

### 2. 八大角色技能
- **许愿**（好人主公）：每轮鉴 2 件兽首
- **方震**（好人老二）：查验玩家阵营
- **姬云浮**（好人）：鉴宝不受老朝奉颠倒影响，被封印则永久失能
- **黄烟烟 / 木户加奈**（好人平民）：每轮鉴 1 件，随机一轮无法鉴宝
- **老朝奉**（坏人主公）：每轮鉴 1 件，可使用「颠倒乾坤」混淆好人鉴宝结果
- **药不然**（坏人老二）：每轮鉴 1 件，可封印一名玩家使其无法行动
- **郑国渠**（坏人小弟）：每轮鉴 1 件，可封存一件兽首使其本轮无法被鉴定

### 3. 三轮鉴宝押币流程
- **鉴宝阶段**：各自运用技能鉴定兽首真伪
- **发言阶段**：按随机顺序依次发言，陈述推断
- **押币阶段**：选择关注的兽首押币，押币最多者隐藏，第二多者揭露
- **揭示阶段**：公布押币结果，揭露兽首真伪，计分

### 4. 胜负判定
- 许愿阵营找出真品 +1 分
- 达到 5 分许愿阵营获胜
- 三轮结束未达目标分数则老朝奉阵营获胜

## 技术架构
- 前端：React 19 + Vite + Tailwind CSS（古风青铜配色）
- 后端：Express + HTTP 轮询（无 WebSocket，规避代理链限制）
- 联机：HTTP 短轮询 + 状态同步
- 部署：Cloud Studio 沙箱环境
